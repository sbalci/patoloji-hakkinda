# Using multiple R versions

#### Backup

```text
tar -cvzf ~/Documents/R-backup.tgz $(ls -d /Library/Frameworks/R.framework/)
```

#### Backup Libraries

```text
tar -cvzf ~/Documents/pkg-backup.tgz $(ls -d /Library/Frameworks/R.framework/Versions/[34]*/Resources/library)
```
